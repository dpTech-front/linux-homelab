# Zsh Configuration

This folder contains my Zsh shell environment setup.

## Included

- **.zprofile** → Automatically starts bspwm/X on tty1  
- **.zshrc** → Interactive shell setup with aliases, functions, and Starship prompt  
- **.zshenv** → Sets `ZDOTDIR` to ~/.config/zsh

## Notes

- Designed for workflow optimization and reproducibility  
- Fully portable across Linux systems  
- Part of my larger Linux home lab setup
