# Linux Home Lab

This repository documents my Linux home lab journey, showcasing configurations, scripts, and projects I’ve built or explored.

## Current Contents

- **shells/** → Shell environments like Zsh, Bash, and Fish  
- **vim/** → Vim / Neovim configuration and plugins  
- **tiling-wms/** → bspwm, Echinus, dwm setups  
- **server-setup/** → Server automation scripts  
- **embedded-robotics/** → Arduino, EV3, and robotics projects

> Each folder is self-contained and documented for step-by-step reproducibility.
