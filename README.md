# Linux Home Lab

This repository contains my **Linux home lab configuration and system setup**, focused on building a modular, efficient, and reproducible working environment.

It reflects my experience working with Linux systems, window managers, and shell environments across multiple distributions.

---

## Background

This repository is a **restructured and documented evolution** of my earlier dotfiles project:

➡️ Original repository:  
https://bitbucket.org/hikiganeZen/dotfiles/src/main/

That repository contains configurations and experiments accumulated over several years (since 2021), during a period where I explored and worked across multiple Linux distributions.

This current repository focuses on:

- Cleaning and standardizing configurations  
- Improving structure and modularity  
- Documenting setups for clarity and reproducibility  

---

## Scope

The project is centered around:

- System configuration and environment setup  
- Tiling window manager workflows  
- Shell customization and productivity  
- Modular and maintainable configuration design  

---

## Components

The repository is organized into independent modules:

### Shell Environment
- Zsh configuration  
- Starship prompt setup  

### Window Management (X11)
- bspwm configuration  
- sxhkd keybindings  
- Multi-monitor handling via `xrandr`  

### UI Components
- Polybar (status bar)  
- Picom (compositor)  

### System Setup
- Modular scripts and configuration structure  
- Portable and reusable across different machines  

---

## Design Principles

- **Modularity** — each component is isolated and reusable  
- **Non-redundancy** — avoids duplication across configurations  
- **Clarity** — structured and documented for maintainability  
- **Portability** — works across different Linux distributions  
- **Scalability** — adaptable for future setups (including Wayland)  

---

## Workflow Focus

This setup is optimized for:

- Keyboard-driven interaction  
- Tiling window manager workflows  
- Efficient navigation and system control  
- Minimal but functional UI
